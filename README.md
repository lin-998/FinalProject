# 项目简介

#### 运行环境
node 11.11.0
HbuilderX
启动项目
git clone https://github.com/cometang/tim-uniapp.git

npm install 

使用HbuilderX打开后即可运行[支持 APP 小程序 h5]

#### 功能简介
* 登录注册
* 即时通讯私聊信息
* 首页轮播图
* 点赞
* 我喜欢的动态
* 图片预览
* 发布页
* 动态详情页
* 图片上传
* 图片压缩(app端)
* 头像背景个人资料更改
